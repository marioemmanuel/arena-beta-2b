NNTP Request Stream

The NNTP Request stream generates a NNTP request header and writes it to the target which is normally a HTWriter stream.

This module is implemented by HTNewsRq.c, and it is a part of the W3C Reference Library. 

#ifndef HTNEWSREQ_H
