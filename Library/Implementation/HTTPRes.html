HTTP Response Stream

The HTTP response stream generates a HTTP response header and writes it to the target which is normally a HTWriter stream.

This module is implemented by HTTPRes.c, and it is a part of the W3C Reference Library. 

#ifndef HTTPRES_H
#define HTTPRES_H

#include "HTStream.h"
#include "HTReq.h"

Streams Definition

This stream makes a server specific HTTP header before it goes into transparent mode. If endHeader is YES then we send an empty CRLF in order to end the header.
