Notes About Solaris for x86

The precompiled binaries of the W3C httpd and its utilities for this platform already do include the follwing patches that were published in the official buglist for the W3C httpd:

Proxy SSL Tunneling
SSL.patch

Patch for chopping off the default ports
HTParse.c.patch2

Date time extension for logfiles
HTLog.c.patch
This bug fix also improved the default configuration.

Core dump patch
HTAccess.c.patch
(Hmm, mine didn't even before applying this patch.)

No automatic lowercase conversion for news messages
HTParse.c.patch

There are no additional requirements for your x86 if you want to run the CERN httpd. If you have any comments on this binary release you're welcome to drop me a note.
