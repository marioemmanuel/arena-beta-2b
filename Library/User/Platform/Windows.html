Notes About Windows

The W3C httpd does not run on windows!

The Library now builds as a set of DLLs! You can not use the Unix makefile for compilation but instead you may use the project files that we provide as part of the distribution files.

We have a set of notes for Windows platform - special thanks to the extraordinary help from:

  • Kok Hoon: Good notes regarding compilation on Windows 3.1 and 95
  • Eric Prud'Hommeaux for creating DLLs and making it compile on all Windows platforms.
  • Charlie Brooks for implementing a new new tread model. See the documentation on this for more
  • Paul Hounslow for initial Windows port
