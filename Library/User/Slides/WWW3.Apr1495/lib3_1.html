New Features in Library 3.1

  • HTTP/1.0 Compliant version
  • Easier to experiment with new HTTP Headers
  • Fully stream oriented architecture
  • Non-blocking interruptible I/O
  • Support for PUT and POST
    • Copy document from one remote server to another
    • Copy local file to remote server
    • Copy memory buffer to remote server
  • Enhanced MIME parser
  • Solid Windows NT support provided by Charlie Brooks, cbrooks@osf.org
  • Ready for release mid June
