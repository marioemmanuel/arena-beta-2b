Slides and Notes on the Library

This is a list of slides and notes from talks on the W3C Reference Library.
WWW2, Chicago October 17, 1994
The first talk about the Libray, an overview of the architecture and what features it contains. It is also a presentation of the libwww thread model that provides thread support.
WWW3, Darmstadt April 14, 1995
Talk about the current status of the Library and HTTP and what new features will include. It includes a plan for how the Library can support collaborative work on the Web.
Advisory Meeting Maj 22 1995
Talk about the 3.1 release of the Library and the plans for new features for the next 6 months.
WWW4, Boston December 13, 1995
Announcement of Library verssion 4.0 which for the first time runs on Windows.

Henrik Frystyk, December 1995, libwww@w3.org
